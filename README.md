## 🧠 What is HireMind AI?

HireMind AI is an **AI-powered career assistant** that simulates how recruiters and ATS systems evaluate candidates.

It helps you:

- 📊 Analyze your resume against multiple job descriptions
- 🎯 Decide which job to apply for
- 🧠 Identify skill gaps
- ✨ Optimize your resume
- 📄 Generate tailored cover letters
- 🎤 Prepare for interviews
- 📌 Track job applications
- 💬 Ask career-related questions via AI

---

## 🔥 Key Features

### ✅ Multi-Job Analysis
- Compare resume with multiple job descriptions
- Get match scores
- Rank jobs by relevance

### ✅ ATS Scoring Engine
- Skills match %
- Keyword match %
- Experience alignment
- Final ATS-style score

### ✅ Skill Gap Detection
- Identify missing skills across jobs
- Prioritize high-impact improvements

### ✅ Resume Optimizer
- Improve bullet points
- Add recruiter-relevant keywords

### ✅ Cover Letter Generator
- Tailored for each job role

### ✅ Interview Preparation
- Role-specific questions
- What interviewers actually evaluate

### ✅ Application Tracker
- Track job status:
  - Not Applied
  - Applied
  - Interview

### ✅ AI Career Chat
- Ask anything about jobs, resumes, or interviews

## 🏗️ Tech Stack

- **Frontend:** Streamlit  
- **LLM:** Groq API (LLaMA 3.1)  
- **Storage:** JSON (No database)  
- **Deployment:** Hugging Face Spaces
